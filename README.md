# SAGE
# Objetivo 
O SAGE (Sistema de Agendamento de Espaços) é uma aplicação desenvolvida para modernizar e otimizar o processo de reserva de ambientes no IFAL – Campus Palmeira dos Índios.
O sistema visa tornar o agendamento mais eficiente, seguro e acessível, permitindo que professores, monitores e equipes administrativas realizem reservas de maneira prática e organizada.
Com isso, busca-se reduzir conflitos de horários, maximizar o uso dos espaços disponíveis e melhorar a gestão das atividades acadêmicas e administrativas.
